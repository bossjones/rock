Rock Better Runtimes

RockStack is collection of packages and a command line tool that allows you to build, test, and run your Node, Perl, PHP, Python, and Ruby apps in a standardized way on CentOS/RHEL, OS X and Ubuntu.

Install

bash -c "$(curl -fsSL https://raw.github.com/rockstack/utils/master/install)"

Create

{% highlight console %}$ rock --runtime=ruby20 init{% endhighlight %} {% highlight console %}$ rock build{% endhighlight %} {% highlight console %}$ cat << EOF > hi.rb require 'sinatra' get '/hi' do "Hello World!" end EOF{% endhighlight %} {% highlight console %}$ rock run ruby hi.rb{% endhighlight %}

Install the packages and read the getting started guides for any one of the supported runtimes.
