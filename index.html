Rock Better Runtimes

RockStack is collection of packages and command line tool that allows you to build, test, and run your Node, Perl, PHP, Python, and Ruby apps in a standardized way on CentOS/RHEL, OS X and Ubuntu.

Install

{% highlight console %}$ bash -c "$(curl -fsSL https://raw.github.com/rockstack/utils/master/install)"{% endhighlight %}

Create An App

{% highlight console %}$ rock --runtime=ruby20 init{% endhighlight %} {% highlight console %}$ rock build{% endhighlight %} {% highlight console %}$ cat << EOF > hi.rb require 'sinatra' get '/hi' do "Hello World!" end EOF{% endhighlight %} {% highlight console %}$ rock run ruby hi.rb{% endhighlight %}
