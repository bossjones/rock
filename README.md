Rock Dev
========

This is the rock development environment.

## Getting Started

 1. Clone the dev repository

        git clone git://github.com/rockstack/dev.git rock

 1. Switch to it

        cd rock

 1. Run the setup script

        ./misc/setup

## Tips

Run a command in all project directories

    ./misc/project git status
